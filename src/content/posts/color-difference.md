## 色差とは

二色間の違いを定義したもの。差が大きいほど区別しやすく、小さいほど区別しにくい。

ユークリッド距離で計算できる。RGB色空間での計算式は以下の通り。

```
distance = √[(R₂-R₁)² + (G₂-G₁)² + (B₂-B₁)²]
```

ただしこれは人間の視覚特性を考慮していないという問題がある。

## LAB色空間とΔE

色差は **ΔE\*** で表される。

知覚できる閾値は 1.0 〜 2.3 の間という研究がある。CIE LABはそれ自体に不均一性があるため、補正された計算式が考案されている。

### 色差の計算式

| 規格 | 概要 |
|------|------|
| **CIE 1976** | CIELAB座標のユークリッド距離を使った初期の計算式 |
| **CIE 1994** | 彩度方向の不均一性を改善 |
| **CIE 2000** | 現在の標準（最新） |
